<h2>Install</h2>
<pre>
git clone https://github.com/brianfiszman/dotfiles
cd ./dotfiles
sudo chmod +x ./install.sh
</pre></br>

Now run the `install.sh` script.

<pre>sh ./install.sh</pre></br>
